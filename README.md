# Proyecto-Personal-Sistema-Gestion-DB-CRUD
Este proyecto es un sistema de escritorio desarrollado en Java para la gestión de clientes, productos y el control de base de datos.

## 📋 Requisitos Previos
Para ejecutar este proyecto, necesitas instalar el siguiente software:

* Apache NetBeans (Versión 12 o superior)
* XAMPP (Para activar el servidor MySQL)
* Java Development Kit (JDK) (Versión 8 o superior)
* MySQL Workbench 8.0 CE (Para ejecutar "BD_script.sql")

## 📁 Estructura del Proyecto
El repositorio contiene los siguientes elementos esenciales:

* 📂 Carpeta del Proyecto: Código fuente listo para abrir en Apache NetBeans.
* 📄 Script SQL: Archivo .sql para crear la base de datos y sus tablas.
* 📦 Librería MariaDB: Archivo .jar necesario para conectar Java con MySQL.

## 🚀 Instalación y Configuración## 1. Configurar la Base de Datos

* Abre el panel de control de XAMPP.
* Inicia los servicios de Apache y MySQL.
* Entra a http://localhost/phpmyadmin.
* Crea una nueva base de datos.
* Importa el archivo .sql incluido en este proyecto.

## 2. Abrir el Proyecto en NetBeans

* Abre Apache NetBeans.
* Ve a File > Open Project.
* Selecciona la carpeta ejecutable del proyecto.

## 3. Vincular la Librería de Conexión

* En NetBeans, ve a la pestaña Projects (lado izquierdo).
* Haz clic derecho sobre la carpeta Libraries de tu proyecto.
* Selecciona Add JAR/Folder.
* Busca y selecciona el archivo .jar de MariaDB incluido en el proyecto.

## 💻 Ejecución

* Haz clic derecho sobre el proyecto en NetBeans.
* Selecciona Run.
* El sistema se iniciará automáticamente.
